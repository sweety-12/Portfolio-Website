## 🛠️ Tech Stack

- **Frontend** — HTML5, CSS3, Vanilla JavaScript
- **Charts** — Chart.js (doughnut chart for DSA progress)
- **Fonts** — Space Grotesk + Inter (Google Fonts)
- **Animations** — CSS keyframes, Intersection Observer, Canvas API
- **Deployed on** — Vercel

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/sweety-12/portfolio.git

cd portfolio

# Just open in browser — no build step needed
open index.html
```

No frameworks. No node_modules. Just open and go. ✨
